F# Editing Support for Open Editors

Emacs

Adds F# mode and F# interactive background process. See the emacs mode in the 'emacs' directory of the repository. Please contribute!

MonoDevelop

  • Code completion
  • Syntax highlighting
  • Tooltips
  • Debugging
  • Target .NET 3.5, 4.0, 4.5
  • F# Interactive scripting (Alt-Enter execution)
  • Templates (Console Application, Library, Tutorial Project, Gtk Project)
  • Makefile support
  • MonoDevelop includes C# 5.0, ASP.NET and other features
  • Supports F# 3.0 type providers (requires F# 3.0)
  • xbuild support for Visual Studio .fsproj and .sln files without change (requires Mono 3.0 and F# 3.0)

Requires MonoDevelop 3.0 or later

Installation (MonoDevelop)

Install the F# Language Binding via the MonoDeveop Add-in manager.

Optionally, get the F# 3.0 compiler. This will enable xbuild/msbuild support for Visual Studio projects.

MonoDevelop --> Add-in manager --> Gallery --> Language Bindings --> F# Language Binding

Then do this:

MonoDevelop --> Options/Preferences --> Source Code --> Code Formatting --> Text File --> Convert tabs to spaces
